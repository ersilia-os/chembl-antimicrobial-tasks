# chembl-antimicrobial-tasks
Get antimicrobial tasks from ChEMBL framed as binary classifications. 

This is the updated version of [chembl-binary-tasks](https://github.com/ersilia-os/chembl-binary-tasks).

## Setup
We recommend creating a Conda environment to run this code:

```sh
conda create -n adda4tb python=3.10
conda activate adda4tb
pip install -r requirements.txt
```
